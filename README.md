This program is a quick and easy parser/sorter of data acquired through Kaggle, specifically used on video game sales.
Some of the functions can be tinkered with to parse different data depending on how your header has the columns sorted.
The intent of this program was to use our toolbox of skills acquired from a junior college CSC-122 class and tackle challenges in a group/partner setting.
The entire program was written in C++ with some aid from AI
